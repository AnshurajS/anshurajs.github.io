A single-page web application designed to visualize and explore exoplanet data. 
An interactive platform that enables users to explore data for over 200 exoplanets, showcasing characteristics such as radius and orbital distance.
A responsive, user-friendly interface and implemented features such as an interactive filtering system and dynamic visualizations with smooth animations. 
Data from NASA’s Exoplanet Archive
